# shotplayer

## 介绍
本地视频播放器，功能：

1. 花样截图，包括：普通截图、GIF截图、字幕截图（本就是觉得115网盘的视频预览降低了画质……所以想找个替代品）

2. 字幕时间偏移调节

3. 默认下载路径设置

4. 选择外挂/内嵌字幕

5. 拖拽添加文件夹/视频

6. 一些快捷键：w/e(调节亮度)、d/f(~~伪~~按帧进退)、x/c(播放速度调节)

## 调试与打包
调试：（gif.worker.js）的路径有问题，导致调试时无法生成gif
```bash
npm run dev
```
打包：
```bash
npm run build:win
```
